Installation instructions for Thinknowlogy 2018r4 (New Science):

• This software uses Java. In order to install or upgrade the Java Runtime Environment, go to http://java.com/ and follow the instructions;
• For Windows 7 and Windows XP, please check if you have downloaded the 7-Zip version;
• Now extract the downloaded Zip file. It doesn't matter in which folder the file is extracted;
• Start Explorer (for files) and go to the extracted folder, called Thinknowlogy2018r4;
• Here you will find Java archive file Thinknowlogy2018r4. Double-click on this file to start the program;
• The documentation and the source code of the program (in Java as well as C++) are included;
• This program applies software license GPLv2.
